## Getting Started

### Clone the repository

```sh
git clone https://github.com/rolandasg/spacex-launches.git
cd spacex-launches
```

### Install dependencies

```sh
npm install
# or
yarn install
```

### Run the app

```sh
npm run dev
# or
yarn dev
```
